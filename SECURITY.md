# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅        |

## Reporting a vulnerability

**Do not open a public issue for security vulnerabilities.**

Use GitHub's private vulnerability reporting:
**[Report a vulnerability](https://github.com/amosjuda/sagaweaw/security/advisories/new)**

Describe the vulnerability, steps to reproduce, and potential impact. You will receive a response within 72 hours. Once confirmed and fixed, you will be credited in the release notes (unless you prefer to remain anonymous).
